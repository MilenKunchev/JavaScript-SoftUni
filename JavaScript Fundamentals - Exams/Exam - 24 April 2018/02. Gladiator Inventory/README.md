<h1 align=center style='text-align:center'>Problem <span lang=BG>2</span> – Gladiator
Inventory</h1>

<p class=MsoNormal>As a gladiator, Pesho has cool Inventory. He loves to buy
new equipment. You are given Pesho`s inventory with all <span lang=EN-GB>of</span>
his equipment -&gt; strings<span lang=BG>,</span> separated by whitespace.
Until you receive <b>&quot;Fight!&quot; </b>you will be receiving commands
which Pesho does with his inventory.</p>

<p class=MsoNormal><span lang=EN-GB>You may receive the following commands:</span></p>

<p class=Code style='margin-left:.5in;text-indent:-.25in'><span lang=EN-GB
style='font-family:Symbol;font-weight:normal'>·<span style='font:7.0pt "Times New Roman"'>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;
</span></span><span lang=EN-GB>Buy {equipment}</span></p>

<p class=Code style='margin-left:.5in;text-indent:-.25in'><span lang=EN-GB
style='font-family:Symbol;font-weight:normal'>·<span style='font:7.0pt "Times New Roman"'>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;
</span></span><span lang=EN-GB>Trash {equipment}</span></p>

<p class=Code style='margin-left:.5in;text-indent:-.25in'><span lang=EN-GB
style='font-family:Symbol;font-weight:normal'>·<span style='font:7.0pt "Times New Roman"'>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;
</span></span><span lang=EN-GB>Repair {equipment}</span></p>

<p class=Code style='margin-left:.5in;text-indent:-.25in'><span lang=EN-GB
style='font-family:Symbol;font-weight:normal'>·<span style='font:7.0pt "Times New Roman"'>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;
</span></span><span lang=EN-GB>Upgrade {equipment}-{upgrade}</span></p>

<p class=MsoNormal><span lang=EN-GB>If you receive <b>Buy command</b>, you
should <b>add</b> the equipment at last position in the inventory, but only if
it isn`t bought already.</span></p>

<p class=MsoNormal><span lang=EN-GB>If you receive <b>Trash command</b>, <b>delete</b>
the equipment if it exists.</span></p>

<p class=MsoNormal><span lang=EN-GB>If you receive <b>Repair command</b>, you
should <b>Repair</b> the equipment if it exists and place it on <b>last
position</b>.</span></p>

<p class=MsoNormal><span lang=EN-GB>If you receive <b>Upgrade command</b>, you
should check if the equipment exists and <b>insert</b> after it the upgrade in
the following format: &quot;<b>{equipment}:{upgrade}&quot;;</b></span></p>

<h2>Input / Consrtaints</h2>

<p class=MsoNormal>You will receive an <strong><span style='font-family:"Calibri","sans-serif"'>array
of strings</span></strong>. Each element of the array is a command.</p>

<p class=MsoListParagraphCxSpFirst style='text-indent:-.25in'><span
style='font-family:Symbol'>·<span style='font:7.0pt "Times New Roman"'>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;
</span></span>In the <strong><span style='font-family:"Calibri","sans-serif"'>first
input element,</span></strong> you will receive Pesho`s <b>inventory</b> – sequence
of equipment names, separated by space.</p>

<p class=MsoListParagraphCxSpLast style='text-indent:-.25in'><span
style='font-family:Symbol'>·<span style='font:7.0pt "Times New Roman"'>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;
</span></span>Each following element, until you receive <b>&quot;Fight!&quot;</b>
will be a <b>command</b>. </p>

<h2>Output</h2>

<p class=MsoListParagraph style='text-indent:-.25in'><span style='font-family:
Symbol'>·<span style='font:7.0pt "Times New Roman"'>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;
</span></span>As output you must print Pesho`s <b>inventory</b>. </p>

<h2>Constraints</h2>

<p class=MsoListParagraphCxSpFirst style='text-indent:-.25in'><span
style='font-family:Symbol'>·<span style='font:7.0pt "Times New Roman"'>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;
</span></span>The<b> command will always be valid.</b></p>

<p class=MsoListParagraphCxSpMiddle style='text-indent:-.25in'><span
style='font-family:Symbol'>·<span style='font:7.0pt "Times New Roman"'>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;
</span></span>The <b>equipment </b>and <b>Upgrade</b> will be strings and will
contain any character, except <b>'-'</b>.</p>

<p class=MsoListParagraphCxSpLast style='text-indent:-.25in'><span
style='font-family:Symbol'>·<span style='font:7.0pt "Times New Roman"'>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;
</span></span>Allowed working <b>time</b> / <b>memory</b>: <b>100ms</b> / <b>16MB</b>.</p>
